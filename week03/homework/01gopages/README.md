# 并发爬取网页图片数量统计工具

## 功能说明
- 并发爬取第8~48个网页（共41个链接）的图片标识
- 统计图片总数（去重和不去重）
- 计算任务总耗时
- 最大并发数限制为5

## 目录结构
01gopages/
── main.go # 入口文件：任务调度、结果统计、耗时计算
── crawler.go # 爬取模块：请求网页、提取图片标识
── utils.go # 工具模块：生成目标链接列表
── README.md # 技术文档

## 核心逻辑
1. **链接生成**：通过循环生成 `https://study-test.sixue.work/html/8.html` 到 `48.html` 的链接
2. **并发控制**：使用信号量（缓冲通道）限制最大并发数为5
3. **图片提取**：通过正则表达式匹配 `图片X` 或 `图片 X` 格式的标识，统一格式后去重
4. **结果统计**：使用互斥锁（sync.Mutex）保护共享变量，确保并发安全
5. **耗时计算**：通过 `time.Now()` 和 `time.Since()` 计算总耗时
