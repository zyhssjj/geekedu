  题库管理系统 - 技术文档

   项目概述
本项目是一个前后端分离的题库生成管理系统，支持AI自动出题、手工出题、题目管理等功能。

   技术栈

    前端技术栈
- **React 18** - 前端框架
- **Zustand** - 状态管理
- **Ant Design 5** - UI组件库
- **React Router 6** - 路由管理
- **Axios** - HTTP客户端
- **React Markdown** - Markdown渲染
- **Vite** - 构建工具

    后端技术栈
- **Go 1.21** - 后端语言
- **Gin** - Web框架
- **GORM** - ORM框架
- **SQLite** - 嵌入式数据库
- **godotenv** - 环境变量管理

   数据库设计

    题目表 (questions)
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | uint | 主键，自增 |
| type | varchar(20) | 题型 |
| content | text | 题目内容 |
| options | text | 选项（JSON格式） |
| answer | varchar(100) | 正确答案 |
| difficulty | varchar(10) | 难度 |
| language | varchar(20) | 编程语言 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |
| deleted_at | datetime | 删除时间（软删除） |

   API接口

    学习心得接口
- `GET /api/note/get` - 获取学习心得

    题目管理接口
- `GET /api/question/list` - 获取题目列表（分页）
- `POST /api/question/add` - 添加单个题目
- `POST /api/question/add-batch` - 批量添加题目
- `POST /api/question/update` - 更新题目
- `POST /api/question/delete` - 删除题目
- `POST /api/question/ai-generate` - AI生成题目
